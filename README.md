# FroBebrik — Мини-LMS (MVP)

Коротко: учебная платформа для изучения Python с встроенной IDE, двумя ролями (преподаватель, студент), загрузкой материалов и системой задач.

Запуск (через Docker Compose):

```bash
docker compose build
docker compose up
```

Сервисы:
- backend: Django REST API (порт 8000)
- frontend: React + Tailwind, собранный и обслуживаемый nginx (порт 5173)
- worker: обработчик отправленных решений (выполняет код в контейнерах, если доступен Docker)

Внимание по безопасности:
- Endpoint запуска кода предназначен только для разработки. В продакшне обязательно использовать изоляцию (контейнеры, ограничение ресурсов, отключение сети, анализ опасного кода).

Дальше можно:
- добавить авторизацию/аутентификацию и разграничение прав (teacher/student)
- улучшить UI календаря и добавить экспорт/импорт материалов
- заменить DB-поллинг worker на очередь задач (Redis/RQ или Celery)
